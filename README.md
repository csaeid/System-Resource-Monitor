# ⭐ Python System Resource Monitor

This project is a terminal-based application written in Python that monitors system resources (CPU, RAM, Disk, and OS information) in real-time.

## ⭐ Features
* **Real-Time Monitoring:** Continuously updates system resource usage.
* **Modular Architecture:** Clean and organized code with separate modules for CPU, RAM, and Disk monitoring.
* **Custom ASCII Art:** Features a unique and fun terminal UI (including Shrek!).

## ⭐ Prerequisites
Before running the application, ensure you have Python installed on your system. You will also need to install the `psutil` library.

## ⭐ How to Run
1. Clone this repository to your local machine:
   ```bash
   https://github.com/csaeid/System-Resource-Monitor.git

2. Navigate to the project directory:
```cd SystemResourceMonitor```

3. Install the required dependencies:
```pip install psutil```

4. Run the main script:
```python main.py```
